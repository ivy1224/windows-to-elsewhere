# 個人品牌網站

這是一個使用 Next.js 和 Tailwind CSS 建立的個人品牌網站，主要分享旅外生活、行銷見解和自我成長的內容。

## 功能特點

- 響應式設計
- 現代化的 UI/UX
- 部落格系統
- 聯絡表單
- 多語言支援（預留）

## 技術棧

- Next.js 14
- React 18
- TypeScript
- Tailwind CSS
- Heroicons

## 開始使用

1. 克隆專案
```bash
git clone [repository-url]
```

2. 安裝依賴
```bash
npm install
```

3. 啟動開發服務器
```bash
npm run dev
```

4. 在瀏覽器中打開 [http://localhost:3000](http://localhost:3000)

## 部署

這個專案可以部署到 Vercel、Netlify 或其他支援 Next.js 的平台。
